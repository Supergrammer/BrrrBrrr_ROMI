# BrrrBrrr Bixby

![빅스비 뱃지](https://img.shields.io/badge/Bixby-8.0.1-informational) 



## Usage

1. 스마트폰의 빅스비를 실행합니다.

2. 부릉부릉 캡슐을 검색 후 추가합니다.

3. 빅스비에게 '부릉부릉 시작'
